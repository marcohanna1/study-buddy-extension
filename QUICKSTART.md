## 🚀 GET IT RUNNING (5 Minutes)

### STEP 1: Download the Project (1 min)

The project is ready in: `/home/claude/study-buddy-extension`

**If you're using VS Code:**
1. Open VS Code
2. File → Open Folder
3. Navigate to where you want to save it
4. Copy the entire `study-buddy-extension` folder there

---

### STEP 2: Install Dependencies (2 min)

Open terminal in the project folder and run:

```bash
npm install
```

**Wait for it to finish** - you'll see something like:
```
added 234 packages in 45s
```

---

### STEP 3: Get FREE API Key (1 min)

While npm is installing, do this:

1. Open: https://aistudio.google.com/app/apikey
2. Sign in with Google
3. Click "Create API key"
4. **COPY IT** → Keep in notepad for next step

---

### STEP 4: Build the Extension (30 seconds)

In terminal, run:

```bash
npm run build
```

You'll see:
```
webpack 5.x.x compiled successfully
```

This creates a `dist` folder.

---

### STEP 5: Load into Chrome (1 min)

1. Open Chrome
2. Go to: `chrome://extensions/`
3. **Turn ON "Developer mode"** (top-right toggle)
4. Click **"Load unpacked"**
5. Select the `dist` folder
6. See the owl icon appear! 🦉

---

### STEP 6: Add API Key (30 seconds)

1. Click the **Study Buddy extension icon** (top-right of Chrome)
2. Paste your API key
3. Click "Save API Key"

## 🆘 Problems?

**npm not found?**
→ Install Node.js: https://nodejs.org/

**Build errors?**
→ Run: `npm install` again

**Owl doesn't appear?**
→ Refresh the webpage after loading extension

**API errors?**
→ Make sure you saved your API key correctly
